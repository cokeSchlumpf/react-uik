# Tag

Tags are simple data displays.

<Canvas>
<Story name="default" args={{ }}>
    <Content>
        <Tag.Tags>
            <Tag>Some Value</Tag>
            <Tag tagColor={ useAppearance().colors.success }>A green tag</Tag>
        </Tag.Tags>
    </Content>
</Story>
</Canvas>

Tags may have labels.

<Canvas>
<Story name="labels" args={{ }}>
    <Content>
        <Tag.Tags>
            <Tag><Tag.Label>Property</Tag.Label>Value</Tag>
        </Tag.Tags>
    </Content>
</Story>
</Canvas>

Tags can be used as toggles.

<Canvas>
<Story name="toggles" args={{ }}>
    <Content>
        <Tag.Tags>
            <Component initialState={ false }>
                {
                    ({ state, setState }) => <Tag.Toggle tagColor={ useAppearance().colors.success } active={ state } onSelect={ setState }>{ state ? <>I'm selected</> : <>Click me</> }</Tag.Toggle>
                }
            </Component>
        </Tag.Tags>
    </Content>
</Story>
</Canvas>
