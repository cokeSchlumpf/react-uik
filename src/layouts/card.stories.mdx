# Card

Cards are a simple content containers with an optional title.

<Canvas>
<Story name="default" args={{ }}>
<Card>
    <Card.Title>Some title</Card.Title>
    <Card.Content>
        Lorem ipsum dolor sit amte.
    </Card.Content>
</Card>
</Story>
</Canvas>

The elevation property creates a shadow around the box which makes it visually elevated.
